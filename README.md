#Advanced Scientific Calculator (C++)

A powerful command-line calculator built in C++ that supports full expression parsing, variables, and scientific functions.

---

## Features

- Full expression parsing  
  Example: `3 + sin(2) * 5`

- Reverse Polish Notation (RPN)  
  Implemented using the Shunting-Yard algorithm

- Variables support  
  Example:
  x = 5
  y = 3
  x + y * 2
  Scientific functions  
- `sin(x)`
- `cos(x)`
- `log(x)` (base 10)
- `ln(x)` (natural log)

-Command history  
- View previous calculations using `history`

- Colored CLI interface (ANSI)

-Error handling  
- Division by zero  
- Undefined variables  
- Invalid expressions
